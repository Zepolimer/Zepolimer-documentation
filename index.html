Introduction

HyperText Markup Langage (HTML) est le code utilisé pour struturer une page web et son contenu. Ce dernier pourra être composé de titres, d'ensemble de paragraphes, de listes, d'images, etc.

HTML n'est pas réellement un langage de programmation, il s'agit plutôt d'un langage de balises qui définissent la struture de la page web. HTML est composé d'une série d'éléments, utilisés pour entourer ou envelopper les diverses parties du contenu, afin qu'elles apparaissent d'une certaines façon.

Les balises entourantes peuvent être rendues par un mot ou une image lien hypertexte vers quelque chose d'autre, la police en italique ou gras, plus grand ou plus petite et ainsi de suite. Par exemple, si l'on souhaite que "mon chat est très grincheux" s'affiche sur la page web, nous indiquerons qu'il s'agit d'un paragraphe en l'entourant des balises paragraphe :

<p>mon chat est très grincheux<p>
Balises

L'élément paragraphe ci-dessus est composé :

  • D'une balise ouvrante, qui se compose du nom de l'élément (ici "p") entre deux chevrons. Cela indique le début de l'élément, ici le début du paragraphe.
  • Le contenu, qui se trouve entre la balise ouvrante et la balise fermante, ici "mon chat est grincheux".
  • La balise fermante, qui se compose de chevrons et du nom de l'élément auquel on rajoute une barre oblique avant le nom de l'élément. Cela indique la fin de l'élément.

    • Les éléments peuvent aussi contenir des attributs <class> ou <div> qui contiennent des informations supplémentaires sur l'élément et que l'on ne souhaite pas afficher avec le contenu. Ces attributs permettent d'identifier l'élément et de l'utiliser pour modifier la mise en forme ou autre.

    Un attribut doit toujours avoir :

  • Un espace entre l'attribut et le nom de l'élément ou l'attribut précédent s'il y en a plusieurs.
  • Un nom (celui de l'attribut) suivi d'un signe égal (=).
  • De guillemets anglais (") pour encadrer la valeur de l'attribut.
    • Imbriquer des éléments

    Vous pouvez placer des éléments au sein d'autres éléments, c'est ce qu'on appelle l'imbrication. Par exemple, si l'on souhaite que le chat soit très grincheux, on place le mot "très" dans un élément <strong> ce qui signifie que le mot sera fortement mis en relief.

    <p>mon chat est <strong>très</strong> grincheux<p>

    Toutefois, il faut faire attention à ce que les éléments soient bien imbriqués les uns dans les autres. Lorsque l'on ouvre deux élements, il est important de les fermer dans l'ordre d'ouverture, comme sur l'exemple ci-dessus. Ainsi, les éléments doivent être ouverts et fermés correctement de façon à ce qu'ils soient correctement à l'intérieur ou à l'extérieur les uns des autres.

    Les éléments vides

    Certains éléments n'ont pas de contenu. Ces éléments sont appelés éléments vides. Prenons l'élément <img> comme exemple :

    <img src="images/firefox-icon.png/>

    Cet élément contient deux attributs mais pas de balises ouvrantes et fermantes. Celles-ci sont remplacées par une balise auto-fermante. L'élement image n'embarque pas de contenu, son but est d'intégrer une image dans la page HTML, à l'endroit où l'élément est placé.

    Structure d'un document HTML

    Pour l'instant, nous avons vu quelques éléments HTML de base. Séparément, ils ne sont pas très utile. En les combinant, on peut créer une page HTML complète.

    <!DOCTYPE html> : Les doctypes étaient utilisés pour dire qu'un document était du HTML valide et détecter les erreurs de balisage.
    <html> </html> : Cet élément encadre tout le contenu de la page, il est parfois appelé élement racine.
    <head> </head> : Contient les mots-clés, description de la page, lien vers les fichiers CSS, etc qui ne sont pas affichés sur la page web.
    <body> </body> : Utilisé pour afficher sur la page web tout le contenu, qu'il s'agisse de texte, images, videos, etc.
    Les titres

    Ils permettent de structurer le contenu en fonctionnant comme dans un livre : du plus important au moins important. Il existe 6 niveaux de titres, de h1 à h6. La plupart du temps, 3 niveaux de titres suffisent amplement :

    <h1>Mon titre principal</h1>
    <h2>Mon titre de section</h2>
    <h3>Mon sous-titre</h3>
    Les paragraphes

    Les paragraphes contiennent du texte et ils sont très utilisés pour créer du contenu. Comme nous l'avons vus plus haut avec l'exemple du chat. On peut utiliser du texte Lorem Ipsum lorsque l'on crée un site web pour s'exercer à la pratique. Il faut toutefois faire attention aux balises, que l'on ne peut pas noter dans un paragraphes pour qu'elles soient affichées. Il est également important de faire attention aux accents et à la ponctuation.

    Les liens

    Les liens sont très importants, ce sont eux qui font que le web est une toile sur laquelle on peut naviguer de page en page. Pour créer un lien, il suffit d'utiliser l'élément <a> qui correspond à ancre.

    <a href="siteweb.com">Site Web !</a>

    Attention à ne pas oublier la partie avec https:// ou http:// qui représente le protocole utilisé. Une fois le lien créé, il est primordial de tester la page en vérifiant que le lien fonctionne.
